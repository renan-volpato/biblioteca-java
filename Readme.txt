Para executar o projeto, siga os passos abaixo:

Pré-requisitos:

-Certifique-se de ter o JDK 8 (ou superior) instalado.
-tenha o MySQL Server rodando na sua máquina. 

Banco de Dados:

-Abra o seu cliente MySQL (Workbench, DBeaver, etc.).
=Execute o script biblioteca.sql disponível na raiz do projeto para criar o banco de dados e as tabelas.

Execução:

-Abra o projeto no VS Code.
-Certifique-se de que a extensão "Extension Pack for Java" está instalada.
-Localize o arquivo Main.java dentro de src/main/java/br/com/biblioteca/.
-Clique com o botão direito sobre ele e selecione "Run".
